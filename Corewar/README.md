<h1>Corewar</h1>

### How to use

`git clone https://github.com/lalves42/42Projects.git; cd 42Projects/Corewar; make bonus; cp Assembler/asm VM/corewar ./`  
Then cp some champions.s from `./champions/...` to `./` and you're ready to go.
